# measure-me-jenkins

Application to extract build health metrics from Jenkins

## Prerequisites
* Python3 installed
    * <https://www.python.org/downloads/>
* virtualenv installed
    * <https://virtualenv.pypa.io/en/latest/>
    * sudo pip3 install --upgrade virtualenv virtualenvwrapper
